Printer Troubleshooting

If you are experiencing any issues with your printing—such as an incorrect printer selected or you forgot your PIN — please follow the troubleshooting steps outlined below.
🔑 Oops, I forgot my PIN code

Forgotten PIN Code?

Visit your school's uniFLOW website

Once the login window appears, simply click the button labeled "Continue with [Your School Name]..

Access your Dashboard

From the menu on the left side of the screen, select the Dashboard option.

Retrieve your PIN

In the Identities section of your dashboard, locate the PIN option. Click on the three-dot menu and select Show PIN code to view your credentials.

🏷️ Check Printer name

Check Printer name

When you are ready to print, a print dialogue window will appear. While the layout of this window may vary slightly depending on your software, it will always allow you to select which printer you wish to use.

Please note that printer names vary from school to school. Common examples:

  • WFS_Print
  • CanonSecurePrint
  • Secure Print

To ensure your document prints correctly, you must select the appropriate printer from your list. If you are unsure which one to choose, the best approach is to check with a colleague. They can open their own print dialogue (typically by using the Ctrl + P shortcut) to verify the exact printer name used at your location.

🔄 uniFLOW app check

Is uniFLOW app run check

To use a cloud-based printer, the uniFLOW app must be running on your device before you print. While it usually starts automatically, you may occasionally need to start or restart it manually.

Checking if uniFLOW is Running: You can verify the app's status by checking the system tray in the bottom-right corner of your screen. If the app is running and connected, the uniFLOW icon will be visible.

If you need to shut the app down for a future restart, simply right-click the icon and select Exit.

How to Start the App

  • Press the Windows Start button (🪟).
  • Type "uniFLOW" to find the app.
  • You will see it listed in the search results or the app list on the left.
  • Click Open to start the application.

Managing Settings

Once uniFLOW is active, you can view your settings by double-clicking the app icon in the system tray. This will open the uniFLOW window, where you can find:

  • Online Service Link: Access the uniFLOW web service if needed.
  • Account Information: Verify that you are logged in under the correct account name.
  • Printer Details: Confirm the name of the printer assigned to your location.

If the app is not connected, you can use the Re-register button located at the bottom of the window to refresh the connection.
